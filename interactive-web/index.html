  • 01 - mouse move 값 활용 : 마우스 위치에 따라서 오브젝트가 움직이게 한다. (모바일은 자이로센서 활용) (새창 열림)
    1. window.addEventListener함수, mousemove 이벤트 사용
    2. 마우스이동시 mousemove 이벤트를 트리거할 때마다 clientX, clientY 속성으로 좌표 확인.
    3. window.onload함수로 시각적으로 값을 확인하기 위한 테스트 코드 삽입 : innerHTML 속성사용
    4. 마우스포인터를 따라다니는 박스를 생성 후 style.transform 속성으로 translate에 트리거한 x, y 좌표 값을 저장

  • 02 - requestAnimationFrame (loop, 자연스러운 움직임) (새창 열림)
    1. 구문 : requestAnimationFrame(callback);
    2. 브라우저에게 수행하고자 하는 애니메이션을 알리고 다음 리페인트가 진행되기 전에 해당 애니메이션을 업데이트하는 함수를 호출하게 한다.
    3. 리페인트 이전에 실행할 콜백을 인자로 받는다.
    4. 보통 1초에 60회 콜백하지만 대부분의 브라우저에서는 W3C 권장사항에 따라 디스플레이 주사율과 일치한다.
    5. 캔버스 애니메이션이나 3D컨텐츠에 많이 사용한다.
    6. 모바일의 자이로 센서에서도 활용 가능.

  • 04 - 3D공간감과 마우스 위치값을 가운데로 주기 (새창 열림)
    1. 브라우저 내에서 마우스 위치값을 중앙으로 오게 한다.
    2. transform-style 속성으로 3D공간에서 렌더링 되는 방식을 지정한다.
