2021.06.19

  1. 자바스크립트란 무엇인가

    • 자바스크립트를 해석하고 실행하는 스크립트 엔진을 가지고있는 대표적인 애플리케이션이 웹 브라우저이다.(chrome, safafi, ie 등등...)
    • 동적으로 컨텐츠를 바꾸거나 UI를 이동시키는 등의 일을 할수 있다.
    • 크롬브라우저에서 사용하는 자바스크립트 엔진을 이용한 nodeJS가 등장.
    • 데스크탑 애플리케이션, 모바일 애플리케이션등도 만들수 있다.
    • 웹 프론트앤드, 백앤드, 데스크탑, 모바일등 다양한 분야에서 사용된다.

  2. Various JavaScript Runtime

    • 각종 웹브라우저, Node.Js, Electron 등등...
    • ECMA에서 표준화 하고 관리하고있다.

  3. 실습을 위한 준비

    • 브라우저(chrome), 서버환경(node.js), 에디터(VSCode)
    • Node.js로 실행하는 코드를 작성해 본다.
    • 로그를 출력하는 간단한 코드를 작성한다.
    • 커맨드라인 툴(git bash)에서 노드 명령어로 해당파일을 실행한다. > node clip3.js
    • 코드가 실행되고 그 결과가 터미널에 출력된다.

  1. Expression Statement

    • 값을 만들어내는 간단한 코드를 표현식 이라고 한다.
    • 값을 만들어내기 때문에 함수의 인자로 사용할 수 있다. : alert(1000 + 100);
    • statement : 하나 혹은 여러 개의 표현식이 모여 문장을 이룬다.
    • 조건문, 반복문도 문장이다.
    • 표현식이 모여서 문장이 되고, 문장이 모여서 프로그램이 된다.
